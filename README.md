
## Create & active venv
```
$ python3.11 -m venv django
$ source ~/django/bin/activate
```

## Install depedencies
```
$ pip install -r requirements.txt
```

## Migration
```
$ python manage.py migrate
```


## Seed data
```
$ python manage.py seedorder
```

## Assignment
```
$ python manage.py select_canceled_orders
```
